HexiBot is a floating AI chatbot plugin that integrates OpenAI’s GPT for conversational assistance. Fully customizable with prompts, media icons, shortcodes, and memory. Feel free to test it out, as we use it on our own site.

== Installation ==

1. Upload the `hexibot-chatbot` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin via the WordPress admin.
3. Go to **HexiBot > License** and enter your Hexibot license.
4. Go to **HexiBot > Settings** to:
– Enter your OpenAI API key.
– Upload/select a bot avatar and chat icon.
– Choose display mode: show on all pages or via ` ` shortcode.
– Enter WhatsApp connection (optional).
– Choose accessibility options.

== Usage ==

– **Auto mode**: The bot floats on all pages.
– **Shortcode mode**: Place ` ` in any post/page to embed it.

== Features ==

– Powered by OpenAI ChatGPT (API key required), uses 3.5 turbo
– Customize AI prompts
– Floating button and animated chat window
– Chat memory saved in browser (localStorage)
– Reset button to clear conversation
– Admin media upload for avatar & button icon
– Customize Initial welcome message
– Typing animation
– Speech to text
– Text to speech
– Customize Delay display of floating button
– Show on every page or on a single using shortcode

== Frequently Asked Questions ==

= How do I get an OpenAI API key? =
Sign up at [https://platform.openai.com/](https://platform.openai.com/) and create a secret API key.

= Will it remember users between visits? =
Only on the same browser—data is stored in `localStorage`.

= Can I style it further? =
Yes! Customize `hexibot.css` or override with theme CSS.
